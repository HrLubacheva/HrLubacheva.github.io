## 1. О проекте

Сайт-визитка карьерного консультанта, HR-эксперта и бизнес-тренера.  
Помогает компаниям находить сотрудников, а специалистам — развивать карьеру.

**Ключевые функции:**
- Калькулятор услуг с корзиной и скидкой 5%
- Карьерный квиз (подбор услуг по ответам)
- Карусель сертификатов
- Формы обратной связи (экспресс-диагностика, быстрый заказ)
- Отправка материалов на email (чек-лист, программа тренинга)
- Telegram-уведомления о новых заявках
- Google Sheets — база данных заявок
- PWA (можно установить на телефон)

## 5. Ключевые зависимости

### Google Apps Script (бэкенд)
- **URL:** задаётся в `00_config.js` → `SCRIPT_URL`
- **Функции:**
  - `doPost(e)` — принимает заявки с сайта
  - `sendTelegramNotification()` — отправка в Telegram
  - `sendAdminEmail()` — отправка на email администратора
  - `appendToSheet()` — запись в Google Sheets

### Telegram
- **Бот:** `@HrLubacheva_bot` (токен в PropertiesService)
- **Канал:** приватный канал для уведомлений (ID в PropertiesService)

### Google Sheets
- **Таблицы:**
  - `Обратная связь` — заявки с формы диагностики
  - `Быстрый заказ` — заявки из калькулятора
  - `Материалы` — запросы на скачивание
  - `Квиз ответы` — прохождения квиза
  - `Логи` — технические логи
  - `Технические ошибки` — критические ошибки

### Переменные окружения (PropertiesService)
| Ключ | Описание |
|------|----------|
| `SPREADSHEET_ID` | ID Google Sheets |
| `TELEGRAM_BOT_TOKEN` | Токен бота |
| `TELEGRAM_CHAT_ID` | ID канала/чата для уведомлений |
| `TELEGRAM_LOG_CHAT_ID` | ID чата для логов |
| `ADMIN_EMAIL` | Email для уведомлений |

## 7. Как добавить нового разработчика

1. **Доступ к репозиторию:**  
   Добавить пользователя в GitHub-репозиторий (Settings → Collaborators)

2. **Доступ к Google Sheets:**  
   Добавить email разработчика в таблицу (Поделиться → Добавить пользователя)

3. **Доступ к Google Apps Script:**  
   Перейти в редактор скрипта → Поделиться → Добавить пользователя

4. **Переменные окружения (настройки скрипта):**  
   Скрипт → Триггеры и переменные → Script Properties → Добавить ключи

5. **Доступ к Telegram боту:**  
   Передать токен бота (или сделать разработчика администратором бота)

## 8. Частые задачи

### Обновить текст на сайте
1. Найти нужный HTML-файл в `components/sections/`
2. Внести правки
3. Пересобрать: `python build.py`

### Добавить новую услугу в калькулятор
1. Открыть `components/common/js/12_services-data.js`
2. Добавить услугу в нужную категорию
3. Пересобрать проект

### Изменить стили
1. Найти нужный CSS-файл в `components/common/css/`
2. Внести правки
3. Пересобрать (если используется минификация)

### Обновить текст в Telegram-уведомлениях
1. Открыть `Code.gs` (Google Apps Script)
2. Найти функции `formatCallbackMessage` / `formatQuickOrderMessage`
3. Внести правки
4. Опубликовать новую версию (Deploy → Manage deployments → Edit → Deploy)

### Заменить PDF/ZIP файлы
1. Загрузить новые файлы в `assets/docs/` с теми же именами
2. Или обновить пути в `00_config.js`

## 9. Важные ограничения и нюансы

| Что | Описание |
|-----|----------|
| **Телефон в Telegram** | Для автодетекта номер должен быть в формате `+79211234567` (слитно) |
| **CORS** | Настроен через `mode: 'no-cors'` в fetch; бэкенд должен отвечать |
| **Кэш дубликатов** | Заявки с одинаковым `requestId` не принимаются в течение 1 часа |
| **Лимиты API** | ip-api.com — 45 запросов/мин (бесплатно) |
| **Telegram длина** | Сообщения длиннее 4000 символов обрезаются |
