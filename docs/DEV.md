## Что внутри проекта?

Сайт собран из небольших кусочков — **компонентов**. Каждый компонент отвечает за свою часть страницы. Они лежат в папке `components/`.

Вот основные файлы и папки:

- `components/` — все части страницы (шапка, герой, услуги, калькулятор, контакты и т.д.)
- `styles.css` — настройки внешнего вида (цвета, шрифты, отступы)
- `build.py` — программа, которая склеивает компоненты в один файл `index.html`
- `server.py` — программа для запуска сайта на вашем компьютере (чтобы видеть изменения сразу)
- `deploy.py` — программа, которая загружает готовый сайт на GitHub (в интернет)
- `.github/workflows/static.yml` — автоматическая публикация на GitHub Pages
- `img.png` — ваше фото на главном экране

## Как запустить сайт на своём компьютере (чтобы посмотреть правки)

1. Откройте папку проекта в терминале (командной строке).
2. Выполните команду:  
   ```bash
   python server.py
   ```
3. В браузере откроется сайт по адресу `http://localhost:8080`.  
   Теперь вы можете редактировать любой компонент, сохранять файл, обновлять страницу в браузере — изменения появятся сразу.

## Как загрузить обновления в интернет (на GitHub Pages)

1. Убедитесь, что вы сохранили все изменения в компонентах.
2. Запустите команду:  
   ```bash
   python deploy.py
   ```
   Программа сделает всё сама:
   - соберёт `index.html` из компонентов
   - добавит изменения в Git
   - сделает коммит с текущей датой и временем
   - отправит всё на GitHub
3. Через 1-2 минуты сайт обновится по адресу `https://hrlubacheva.github.io/`

## Что делать, если вы не программист, но хотите что-то поменять?

- **Текст** — откройте нужный компонент в папке `components` (например, `hero.html` для главного текста, `contacts.html` для контактов) и просто замените слова.
- **Цены в калькуляторе** — откройте `calculator.html`, найдите нужную услугу и измените число в атрибуте `value="..."`.
- **Изображение** — замените файл `img.png` в корне на своё фото (сохраните то же имя и расширение).
- **Стили (цвета, размеры)** — откройте `styles.css`, в самом начале найдите переменные (`--primary`, `--bg`, `--text`) и поменяйте значения.

## Если что-то пошло не так

- Проверьте, что все файлы сохранены.
- Убедитесь, что у вас установлен Python (версия 3.6 или выше). Команда `python --version` покажет версию.
- При запуске `deploy.py` Git может попросить представиться — выполните один раз:
  ```bash
  git config --global user.name "Ваше Имя"
  git config --global user.email "ваш_email@example.com"
  ```
