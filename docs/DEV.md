## Как запустить локально

```bash
python server.py
```
Сервер запустится на `http://localhost:8080` (или следующем свободном порту). При каждом обновлении страницы `index.html` будет пересобираться из компонентов.

## Как задеплоить на GitHub Pages

```bash
python deploy.py
```
Скрипт выполнит:
1. Сборку `index.html`
2. `git add .`
3. `git commit` с автоматическим сообщением (дата+время)
4. `git push`

После этого GitHub Actions (`.github/workflows/static.yml`) задеплоит сайт.
